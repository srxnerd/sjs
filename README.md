## Js File Scanner Endpoint
- how to use
```bash
    $ git clone https://github.com/srxnerd/sjs.git
    $ cd sjs
    $ go mod init sjs
    $ go mod download github.com/pkg/profile
    $ go build ./sjs.go
    $ ./sjs -help
```
- for example use
```bash
    $ ./sjs -url https://bitcoin.org -api https://bitcoin.org

```
